# Sm808

## Usage

run as ``` bundle exec rspec spec/lib/song_spec.rb```


Play the rake tasks , patterns 8 steps each

```bundle exec rake sm808:play```


And patterns 8, 16, 2 steps each

```bundle exec rake sm808:play2```
